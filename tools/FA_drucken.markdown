## Benutzung
Den folgenden Link einfach im Browser in die Lesezeichenleiste ziehen.

In Catkin dann die Filter so setzen, dass nur die Touren angezeigt werden, die man drucken (bzw. herunterladen) will.
Anschließend auf das gespeicherte Bookmark klicken und es werden alle Fahraufträge heruntergeladen.
